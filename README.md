# pbl3-sales-management
- PBL3 - Công nghệ phần mềm - Ứng dụng quản lý bán hàng điện gia dụng
- Thành viên:
  + Nguyễn Đăng Kiệt (nhóm trưởng)
  + Huỳnh Nhật
  + Hồ Văn Nam
  + Vũ Thị Nga
- Ngôn ngữ lập trình: C#
- Framework: Entity Framework
- Database: MSSQL Server
- Các chức năng của dự án:
  + Đăng nhập, Quên mật khẩu, Đổi mật khẩu, phân quyền (admin, nhân viên)
  + Quản lý nhân viên, khách hàng, nhà cung cấp, hàng hoá, hoá đơn bán/nhập hàng
  + Hoá đơn bán, nhập hàng
  + Báo cáo, thống kê
